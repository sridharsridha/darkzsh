# Dotfiles

ZSH Configuration for UNIX and OSX,

## Installation

Clone this repro
```sh
git clone --depth=1 https://github.com/sridharsridha/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install.py
```
That all, start zsh, oh-my-zsh will install automatically.

## Uninstallation
```sh
rm -rf ~/.dotfiles
rm ~/.zshrc
rm ~/.zsh_modules
```
